# new-avallen

A practice project. It is supposed to store and be able to display information regarding game world, specifically
tied to a DnD west marches campaign.

# Requires

<ul>
    <li>Maven v3.0+</li>
    <li>MySql</li>
    <li>Lombok plugin</li>
    <li>PlantUml plugin</li>
</ul>
